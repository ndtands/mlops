# App predict Price of old Car

## Command useful

```
# Create image
make build_image

# run container
make run_server
```

Open new terminal and run

```
make app
```

## Docker-compose for App

```
# Buiding Docker file
make build

# Start app
make start-app

#Stop app
make stop-app
```
